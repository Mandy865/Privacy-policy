## Information We Collect
We do NOT collect personal information such as:
- Name
- Email
- Phone number
- Address

However, our app uses Firebase Crashlytics (Google) to monitor crashes and improve app stability.

### Automatically collected data
When a crash occurs, Firebase Crashlytics may collect:
- Device model
- Operating system version
- App version
- Crash logs
- Performance and diagnostic data
- Anonymous device identifiers

This information does not personally identify you.
